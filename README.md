Игра находится в активной разработке, поэтому в ней могут встречаться баги, визуальные недочёты, проблемы с балансом и нестабильные механики.

Чтобы запустить игру:

1. Скачайте архив с нужной версией.(Готовые версии игры можно скачать во вкладке **Releases**)
2. Распакуйте `.zip`.
3. Откройте распакованную папку.
4. Запустите `KeySurvival.exe`.

Важно: не запускайте игру прямо из архива. Сначала обязательно распакуйте файлы.

Если Windows покажет предупреждение “Windows защитила ваш компьютер”, нажмите:

**Подробнее → Выполнить в любом случае**

Это появляется потому, что игра пока не имеет цифровой подписи издателя.

---

**Key Survival** — это 2D survival-игра, где игрок появляется в случайно сгенерированном мире, собирает ресурсы, крафтит предметы, строит базу, добывает еду, переживает дни и ночи, сражается с врагами и готовится к событию с мини-боссом.

Проект находится в активной разработке.

---

Цель игры

Основная цель — выжить, построить базу, подготовиться к опасности и победить мини-босса.

Базовая игровая петля:

собрать ресурсы → сделать верстак → построить базу → добыть еду → пережить ночь → получить ключ → открыть лут-сундук → усилиться → победить босса
